# Publiskā saite / statiskā hostēšana

Projekta mape: **`procesu registrs`**.

Lai `index.html` strādātu pilnībā, **tajā pašā mapē** jābūt visiem šiem failiem:

- `index.html`
- `DB.js`, `GP kataogs.js`, `Filtrs.js`, `Procesu registrs.js`, `Skata zona.js`, `Pievienot jaunu procesu.js`, `Procesa kartina.js`, `Izmainu_pieteikums.js`, `Skaidrojumi.js`

HTML `index.html` izmanto URL kodējumu (`%20`) atsaucēm uz failiem ar atstarpēm nosaukumā.

Supabase atslēga ir `DB.js` — nepublicējiet servisa atslēgu publiskā repozitorijā.
